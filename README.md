# Todo管理ツール

プロジェクト管理とタスク追跡のためのWebアプリケーションです。

## 機能

- タスクの階層管理（親子関係）
- 進捗度の自動計算
- 優先度の自動計算
- 担当者管理
- リアルタイム更新
- レスポンシブデザイン

## 技術スタック

- React 18
- TypeScript
- Vite
- CSS Modules

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# GitHub Pagesへのデプロイ
npm run deploy
```

## 使用方法

1. ログイン画面で担当者を選択または登録
2. メイン画面でタスクの作成・管理
3. タスクの編集・進捗更新
4. 担当者設定の管理
